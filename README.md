## ✨ What I'm Working On

<table>
  <tr>
    <td width="50%" valign="top">

### 🧠 PLCPilot
Offline AI fault diagnosis system for PLC troubleshooting using compact language models, hybrid retrieval, and local deployment.

**Highlights**
- Quantized LLM workflow
- Hybrid RAG with FAISS + BM25
- Brand-aware industrial diagnostics
- Safety-aware response generation

  </td>
  <td width="50%" valign="top">

### 👁️ SeeFresh
Accessibility-focused AI web app that helps visually impaired users identify fruit and vegetable ripeness in real time.

**Highlights**
- YOLOv8-based detection
- InceptionV3 classification
- Smartphone-friendly use case
- Accessibility-first idea

  </td>
  </tr>
  <tr>
    <td width="50%" valign="top">

### ♻️ Smart Bin System
Edge AI smart waste management system using embedded vision, IoT sensors, and real-time monitoring.

**Highlights**

- Raspberry Pi deployment
- TensorFlow Lite optimization
- Sensor-based bin monitoring

  </td>

Article-based chatbot for question answering and revision support using LangChain and vector search.

- FAISS vector indexing
- Interactive Q&A and MCQs
- Streamlit-powered interface
